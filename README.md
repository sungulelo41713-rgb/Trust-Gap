# Event Trust & Accountability Platform

## The Problem
The event planning industry suffers from a "Trust Gap." Clients have no verified way to vet suppliers, and suppliers have no protection against unrealistic client expectations or budget disputes. 

## The Solution
This platform introduces an **Accountability Engine** that links every review to a verified service contract. 
* **Verified Reviews:** Only clients linked to a specific Event ID can submit a review.
* **Dual-Feedback Loop:** Both clients and suppliers have a stake in their reputation.
* **Budget Transparency:** Standardized metrics (Budget Adherence, Punctuality) replace subjective complaints.

## Tech Stack
* **Database:** PostgreSQL (via Supabase)
* **Backend:** Python/FastAPI (API Logic)
* **Frontend:** React/Next.js (Dashboard)

## Key Features
- [ ] **Contract Hooks:** Reviews are cryptographically/logically linked to event contracts.
- [ ] **Accountability Dashboard:** View past events and manage pending reviews.
- [ ] **Service Provider Profiles:** Aggregated ratings based on verified data.
